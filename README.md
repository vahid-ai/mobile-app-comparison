# mobile-app-comparison

Comparing different frameworks for building mobile apps.

## zero-native

The first sample targets [zero-native](https://zero-native.dev/) — Zig native shell + web UI, with mobile embedding via `libzero-native.a` and platform host apps.

See [samples/zero-native/README.md](samples/zero-native/README.md) for setup and commands.

Quick start:

```bash
npm install
npm run zn:test
npm run zn:android:apk
```
